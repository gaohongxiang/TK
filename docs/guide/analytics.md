# 数据分析

数据分析用于本地导入 TikTok Shop 商品流量详情 Excel，快速看渠道表现、商品排行、流量漏斗和商品诊断。

这个模块不需要连接 Firebase。Excel 文件只在当前浏览器里读取和解析，不上传到 Cloudflare，不写入 Firestore，也不会持久化到 localStorage。

## 支持的文件

- TikTok Shop 导出的商品流量详情 Excel。
- 文件后缀支持 `.xlsx` 和 `.xls`。
- 表格中需要包含 `ID`、`商品`、`GMV`、`成交件数`、`订单数` 等商品流量字段。

如果页面提示没有找到表头，通常是因为导出的表不是商品流量详情，或表格结构被手工改过。

## 怎么使用

1. 打开主站的 `数据分析`。
2. 点击 `选择 Excel 文件`。
3. 选择 TikTok Shop 导出的商品流量详情表。
4. 页面会在本地解析文件，并生成 KPI、渠道 GMV、流量漏斗、Top 商品和商品明细。

刷新页面后，当前分析结果会消失。需要再次查看时，重新选择 Excel 文件即可。

## 指标口径

- `GMV`：读取表格里的商品 GMV。
- `成交件数`：读取商品成交件数。
- `订单数`：读取商品订单数。
- `曝光`：商城、视频、商品卡、直播等渠道曝光加总。
- `浏览`：各渠道页面浏览次数加总。
- `去重客户数`：各渠道去重客户数加总。
- `CTR`：浏览次数 ÷ 曝光次数。
- `转化率`：成交件数 ÷ 去重客户数。

渠道维度当前覆盖：

- 商城
- 视频
- 商品卡
- 直播

## 商品诊断标签

系统会按 GMV、订单数、曝光、浏览、CTR 等指标给商品生成运营动作标签。

- `爆品放大`：GMV 或订单数较高，优先补库存、加素材、配优惠活动。
- `加流量`：已有成交但曝光不足，优先补短视频和商品卡曝光。
- `换素材`：曝光不低但点击弱，优先换主图、标题关键词和视频前三秒。
- `承接差`：有浏览无成交，检查价格、评价、详情页卖点和优惠券。
- `观察下架`：低曝光且无成交，短期补素材测试，仍无起量则下架或换品。
- `常规观察`：暂不需要特殊动作，继续监控。

这些标签是运营辅助判断，不是平台官方结论。真正决策时还要结合库存、利润、类目周期、达人素材和活动节奏。

## 和商品管理的关系

数据分析不会自动写入商品管理。分析结果只留在当前页面内存里。

如果你想把分析结论沉淀成商品动作，需要手动回到商品管理或订单管理里调整商品信息、SKU 参数、库存计划或运营备注。

## 数据边界

- 不上传 Excel。
- 不保存 Excel 原始文件。
- 不把解析结果写入 Firestore。
- 不把分析结果发送到第三方接口。
- 不依赖 Firebase 连接。

这个模块适合临时复盘一段时间的商品流量表现，不适合作为长期数据仓库。
