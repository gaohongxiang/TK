# 订单管理

订单管理用于记录采购、物流、入仓进度，并汇总销售额、支出和预估利润。

## 新增订单时，推荐怎么录

1. **先填基本信息：**
   - `账号`
   - `下单时间`
   - `订单号`
   - `采购日期`
2. **再填订单明细项：**
   - 逐条添加商品 / SKU / 数量
   - 每条订单明细对应一个商品的一个 SKU
3. **再看订单级汇总：**
   - `总件数`：按所有明细数量自动求和
   - `总重量`：按所有明细 `单件重量 × 数量` 自动汇总
   - `总尺寸`：按实际打包情况自己调整
   - `预估总海外运费`：按当前的 `总重量 + 总尺寸` 自动计算
4. **再填订单级金额：**
   - `总售价（日元）`
   - `达人佣金率（%）`
   - `达人佣金（元）`
   - `总采购额（元）`
   - `是否退款`
5. **最后补订单级状态：**
   - `订单状态`
   - `订单预警`

## 订单明细怎么理解

一张订单里可以有多条明细，每条明细代表：

- 一个商品的一个 SKU
- 这个 SKU 的数量
- 这条明细自己的快递公司和快递单号

## 多条明细的快递信息怎么录

- 每条明细都可以填自己的 `快递公司 / 快递单号`
- 如果多个商品共用同一个单号，可以先把第一条填好，再点 `快递单号` 旁边的复制图标
- 复制后，去另一条明细里正常粘贴单号即可；快递公司会继续按单号自动识别

## 订单里的金额口径

- `总售价`：日元
- `达人佣金率`：百分比
- `达人佣金`：人民币
- `总采购额`：人民币
- `预估总海外运费`：人民币
- `预估利润`：人民币

预估利润公式：

`预估利润 = 实收金额（日元）÷ 汇率 -（总采购额 + 预估总海外运费 + 达人佣金）`
