# 选品原则

这部分按《TikTok 日本跨境选品逻辑》整理。核心不是赌爆款，而是先保证合规、可发货、能复用、还有利润。

## 选品总原则

- **优先轻小件：** 轻、小、好压缩、好打包的商品优先，利润空间通常更稳。
- **先过合规：** 无液体、无品牌、无 IP、无电池、无敏感元素，先把高风险排掉。
- **先看货源：** 优先确认 1688 供应商支持一件代发，避免压货。
- **优先稳定长期：** 优先做稳定、可长期卖的基础品，不盲目追热品。

## 强制规范

- **液体：** 不做。物流禁运、清关风险高。
- **插电产品：** 不做。日本 110V 和国内 220V 不同，兼容问题大。
- **USB 供电产品：** 可以做。5V USB 供电相对更稳。
- **带电池产品：** 不做。尤其是锂电池、充电宝类。
- **品牌 / Logo / IP：** 严格避开，包含品牌侵权、版权和外观专利。
- **商品尺寸上限：** 建议控制在 `60 × 50 × 40 cm` 以内。
- **商品重量上限：** 建议控制在 `30 KG` 以内。
- **尖锐物品：** 谨慎，部分刀具和尖锐品有进口限制。
- **推荐关键词：** `轻、小、无液、无品牌、无IP`，符合越多越优先。

## 品类判断

- **放心做：** 运动与户外、五金工具、家纺布艺、宠物用品。
- **部分可做：** 时尚配件、3C 数码、居家日用、厨房用品、家装建材、电脑办公、箱包、鞋靴、家电、汽车与摩托车、儿童时尚。
- **谨慎或不做：** 家具、图书杂志音频、男女装、手机手表、美妆个护、母婴、食品饮料、玩具、穆斯林时尚。

## 推荐优先级

::: info S级优先做
运动与户外、家纺布艺、五金工具。以轻小件、无液体、无高风险品为主。
:::

::: info A级重点布局
宠物用品、时尚配件、居家日用、电脑办公。前提是无品牌、无 IP、无插电风险。
:::

::: info B级谨慎尝试
厨房用品、儿童时尚、DIY 创意类。能做，但更依赖你对合规、物流和侵权的判断。
:::

## 推荐品类参考

- **运动与户外：** 跳绳、瑜伽垫、弹力带、护膝、健身手套。
- **家纺布艺：** 抱枕套、收纳袋、浴室地垫、桌布。
- **五金工具：** 螺丝刀套装、固定夹、无品牌小工具。
- **宠物用品：** 宠物梳子、玩具球、牵引绳、喂食器。
- **时尚配件：** 发圈、帽子、无品牌太阳镜、腰带。
- **居家日用：** 收纳盒、挂钩、硅胶厨具、浴室置物架。
- **电脑办公：** USB 集线器、鼠标垫、桌面收纳、手机支架。
- **DIY 创意类：** 拼豆、捏捏乐、模具、贴画、拼图，前提是避开版权图案。

## 避坑清单

- **高竞争低利润：** 钥匙扣、手机壳、钢化膜、纸制品。
- **侵权高风险：** 大牌仿品、动漫卡通人物 IP、带品牌 Logo 的商品。
- **清关 / 物流高风险：** 液体、假花、动植物、易燃易爆品、带电池产品、插电电器。
- **政策敏感：** 带国旗商品、部分文化敏感品。
- **售后高风险：** 男女装、尺码依赖强的服饰类。
- **季节性过强：** 节日装饰类，库存风险大。
