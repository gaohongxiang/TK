核心输入

人民币到手价 ÷ 采购价，例如 1.4 表示到手价 = 1.4 × 采购价
以该档位为目标利润率的基准来反推原价
⚙️ 全局参数（平台手续费 / 汇率 / 运费 / 折扣档位）
支持填 0.38 或 38%；任意增删档位都可以。

各折扣档位定价 / 利润一览

折扣 日元售价 人民币到手 利润率
◇ 公式
日元售价 = 原价 × 折扣 × (1 − fee)
人民币到手 = 日元售价 ÷ rate − ship
利润率 = 人民币到手 ÷ 采购价
原价反推 = (采购价 × 目标利润率 + ship) × rate ÷ [基准折扣 × (1 − fee)]

连接你的 GitHub

订单数据保存在你自己 GitHub 账号下的私有 Gist 里，别人看不到，跨设备也能同步。 首次使用需要生成一个仅带 gist 权限的 Token。

  1. 打开 GitHub Token 生成页（已预填权限）
  2. 确认只勾选 gist，有效期按需选（建议 90 天及以上）
  3. 点底部 Generate token，复制 ghp_ 开头的 Token 粘贴到下方
  4. 首次会自动创建私有 Gist；如果你已经有数据 Gist，把它的 ID 填到"可选"栏即可
