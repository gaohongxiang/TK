隐私与数据边界

TK 电商工具箱是静态工具站。本站提供页面、计算、分析和本地交互能力，不保存你的商品、订单或 Excel 原始文件。

数据保存位置

数据类型 保存位置 说明
商品、SKU、账号 你自己的 Firebase Firestore 商品管理保存和读取；本站不保存副本。
订单、订单明细、物流信息 你自己的 Firebase Firestore 订单管理保存和读取；Firestore SDK 负责离线缓存和同步。
商品流量 Excel 当前浏览器内存 数据分析只在本地解析，不上传，不写入数据库。
汇率、运费倍率、连接配置 当前浏览器 localStorage 仅用于本地工具体验，换浏览器需要重新配置。
HTML、CSS、JS、图片、文档 Cloudflare Pages Cloudflare 只托管静态资源。

不会做的事

  • 不会把你的订单或商品保存到工具作者的数据库。
  • 不会上传或持久化 TikTok Shop 商品流量 Excel。
  • 不会把 Cloudflare Workers 当作用户业务数据库。
  • 不会在当前正式版本推荐 Supabase 作为活跃数据源。

Firebase 连接

商品管理和订单管理共用一次 Firebase 连接。连接配置保存在当前浏览器本地，业务数据写入你自己的 Firestore 项目。退出数据库连接后，本地连接配置会被清除。

自己部署

你可以把主站和文档站部署到自己的 Cloudflare Pages 项目。部署后仍然遵守同样的数据边界：Pages 只发布静态文件，业务数据仍然进入用户自己的 Firebase 或浏览器本地内存。
