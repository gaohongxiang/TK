# 产品属性填写规则

## 通用原则

- 必填属性必须填写。
- 有页面原始信息或图片能确认的，按真实信息填写。
- 不确定时优先选择 `その他`、`該当なし`、`なし`、`ノーブランド` 等页面允许的中性选项。
- 如果必填属性会影响合规、材质、适用人群、安全认证或商品真实性，无法确认就停止让用户确认。

## 常见属性

- ブランド：无品牌 / No Brand / ノーブランド。
- 素材：按图片和标题判断；不确定不要编造高端材质。
- カラー：按变种或图片填写。
- サイズ：只有页面已有尺寸或变种明确时填写。
- 対象性別：非穿戴类优先 ユニセックス / 男女兼用 / 該当なし。
- 対象年齢：非儿童商品不要选择儿童/婴儿。
- 原産国：页面有来源才填；不确定选 その他 或停止确认。
- セット内容 / 個数：按真实售卖数量填写。

## 高风险属性

这些属性不能猜：

- 医療機器、認証、規格、PSE、食品衛生、薬機法相关字段。
- 婴儿/儿童适用年龄。
- 皮肤接触安全、抗菌、防水等级、防火等级。
- 材质为贵金属、天然石、玻璃、陶瓷等高风险表述。
