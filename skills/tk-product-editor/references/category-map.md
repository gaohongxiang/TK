# TikTok 日本站分类和属性映射

这是可持续补充的映射表。页面分类名称可能随店小秘/TikTok 更新而变化，执行时以页面实际可选项为准。

## 轻小运动配件

适用：弹力带、护腕、护膝、健身手套等。

- 优先分类：运动与户外相关配件 / トレーニング用品 / フィットネスアクセサリー。
- 常见属性：用途=トレーニング、対象=ユニセックス、素材=页面可判断的材质；不确定时选 その他 或停止确认。

## 布艺收纳和家居小件

适用：布艺收纳袋、防尘罩、抱枕套、小号整理袋。

- 优先分类：ホーム収納 / 収納用品 / ホームテキスタイル。
- 常见属性：素材=ポリエステル/不織布/布；用途=収納；数量按页面真实信息。

## 清洁替换件

适用：清洁布、刷头、拖把替换布、非液体清洁辅助件。

- 优先分类：掃除用品 / クリーニング用品 / 家庭用品。
- 常见属性：用途=掃除、素材=マイクロファイバー/プラスチック等。

## 桌面小收纳和理线

适用：理线夹、桌面小收纳、非电子支架。

- 优先分类：ホームオフィス / デスク収納 / ケーブル管理。
- 常见属性：用途=整理収納、素材=プラスチック/シリコン等。

## 宠物轻小用品

适用：宠物梳、拾便袋、轻小清洁工具。

- 优先分类：ペット用品 / グルーミング用品 / ペット清掃用品。
- 排除：宠物食品、补充剂、药品、喷雾、湿巾、香水、宠物床、大垫子。

## 发饰和布艺配饰

适用：发圈、发夹、布艺配饰。

- 优先分类：ヘアアクセサリー。
- 排除：吊坠、钥匙扣、金属链扣、贵金属、明显首饰属性。

## 未命中映射

如果商品没有命中以上映射：

1. 先按 `selection-rules.md` 判断是否能做。
2. 能做时，根据页面分类搜索商品核心日文词。
3. 选择最贴近实际用途、最低监管风险的分类。
4. 把新映射补进本文件，便于后续复用。
